# open-source-audit Project

## Software Chosen
Git

## About
This project is part of my coursework where I studied an open-source software in detail. I explored its origin, licensing, Linux implementation, and comparison with proprietary alternatives.

## Contents
- Project Report
- Shell Scripts (script1.sh to script5.sh)
  
## Scripts Description
- script1.sh: Displays system information
- script2.sh: Checks Git installation and version
- script3.sh: Shows disk usage and permissions of directories
- script4.sh: Analyzes a log file and counts error lines
- script5.sh: Generates a personalized open source manifesto


## How to Run Scripts
chmod +x script1.sh  
./script1.sh
